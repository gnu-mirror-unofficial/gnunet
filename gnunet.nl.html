Inleiding

GNUnet is een raamwerk voor beveiligd peer-to-peer netwerken. De voornaamste toepassing geïmplementeerd binnen het GNUnet raamwerk is anonieme censuur-resistente file-sharing.

Het doel van het GNUnet project is het bieden van een infrastructuur voor een veilig Peer-to-peer netwerk. Alle communicatie in GNUnet is ge-authentificeerd en computer-tot-computer geencrypteerd. The economic model makes attacks on the network harder since the economics can be used to control resource usage. GNUnet peers exchange messages using a pluggable transport service abstraction. Momenteel zijn er transport-diensten gebaseerd op UDP, TCP, HTTP en SMTP beschikbaar. The GNUnet core provides mechanisms to perform resource allocations for CPU, bandwidth and storage space. The core enforces resource limitations set by the user. GNUnet does not rely on any centralized services. New Peer-to-Peer protocols can be easily implemented on top of the basic GNUnet infrastructure. Current protocols include anonymous file sharing, a trivial chat protocol, message throughput benchmarking and network topology visualization.
While our goals are similar to projects like Freenet, Gnutella, Mnet, and others, we hope to provide a superior combination of features for users that value security more than efficiency.

Een lijs tvan Veel Voorkomende Vragen is beschikbaar. A couple of papers may give some further insight. Extensive documentation can be found on ovmj.org.

Downloading GNUnet

GNUnet can be found on ftp://ftp.gnu.org/gnu/gnunet/, the GNU mirrors or on ovmj.org.

Vereisten:
  • GNU/Linux, FreeBSD, OpenBSD, NetBSD or Solaris.
  • gtk+ (>= 1.2)
  • libextractor (>= 0.3.0)
Hou er aub rekening mee dat dit een beta-versie is. There may still be some stability, scalability and performance issues.
Totnutoe is het netwerk zeer klein. Er is nog maar weinig content beschikbaar. Het kan een goed idee zijn om u in te schrijven in de GNUnet mailinglijst.
