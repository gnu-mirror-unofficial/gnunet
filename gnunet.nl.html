Inleiding

GNUnet is een raamwerk voor beveiligd peer-to-peer netwerken. De voornaamste toepassing geïmplementeerd binnen het GNUnet raamwerk is anonieme censuur-resistente file-sharing.

Het doel van het GNUnet project is het bieden van een infrastructuur voor een veilig Peer-to-peer netwerk. Alle communicatie in GNUnet is ge-authentificeerd en computer-tot-computer geencrypteerd. The economic model makes attacks on the network harder since the economics can be used to control resource usage. GNUnet peers exchange messages using a pluggable transport service abstraction. Momenteel zijn er transport-diensten gebaseerd op UDP, TCP, HTTP en SMTP beschikbaar. The GNUnet core provides mechanisms to perform resource allocations for CPU, bandwidth and storage space. De kern dwingt bronlimitieten af, ingesteld door de gebruiker. GNUnet is niet afhankelijk van gecentraliseerde diensten. Nieuwe peer-to-peer protocols kunnen gemakkelijk worden geïmplementeerd, aanvullend bij de standaard GNUnet-infrastructuur. Current protocols include anonymous file sharing, a trivial chat protocol, message throughput benchmarking and network topology visualization.
Terwijl ons doel gelijkaardig is aan dat van projecten zoals Freenet, Gnutella, Mnet, en anderen, hopen wij om een superieure combinatie van functies aan te bieden voor gebruikers die meer waarde hechten aan veiligheid dan aan efficiëntie.

Een lijs tvan Veel Voorkomende Vragen is beschikbaar. Een aantal papers kunnen verdere inzichten verschaffen. Uitgebreide documentatie kan op ovmj.org gevonden worden.

Download GNUnet

GNUnet kan gevonden worden op ftp://ftp.gnu.org/gnu/gnunet/, de GNU mirrors of op ovmj.org.

Vereisten:
  • GNU/Linux, FreeBSD, OpenBSD, NetBSD or Solaris.
  • GNU MP Bignum Library (>= 4.0.0)
  • libgcrypt (>= 1.2.0)
  • gtk+ (>= 2.0)
  • libextractor (>= 0.3.3)
Hou er aub rekening mee dat dit een beta-versie is. Er kunnen nog steeds stabiliteits-, schaal- en performancemoelijkheden zijn.
Totnutoe is het netwerk zeer klein. Er is nog maar weinig content beschikbaar. Het kan een goed idee zijn om u in te schrijven in de GNUnet mailinglijst.
