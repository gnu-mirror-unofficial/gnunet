Introduktion

GNUnet är ett ramverk för säker "peer-to-peer" kommunikation. The primary application implemented within the GNUnet framework is anonymous censorship-resistant file-sharing.*

Målet med GNUnet projektet är att skapa en infrastruktur för säker "peer-to-peer" kommunikation. All kommunikation i GNUnet är authenticerad och länk-till-länk-krypterad. Den ekonomiska modellen gör att attacker på nätverket svårt, då ekonomifunktioner kan användas för att kontrollera resursanvändning. GNUnet peers exchange messages using a pluggable transport service abstraction.* Currently, transport services based on UDP, TCP, HTTP and SMTP are available.* The GNUnet core provides mechanisms to perform resource allocations for CPU, bandwidth and storage space.* The core enforces resource limitations set by the user.* Gnunet är inte beroende av någon central server. Nya "peer-to-peer"-protokoll kan enkelt implementeras ovanpå GNUnets grundläggande infrastruktur. Nuvarande protokoll omfattar anonym fildelning, ett enkelt chattprotokoll, mätning av genomströmningsprestanda och visualisering av nätverkstopologin.
While our goals are similar to projects like Freenet, Gnutella, Mnet* , and others, we hope to provide a superior combination of features for users that value security more than efficiency.*

En lista på vanligt frågade frågor (FAQ) finns tillgänglig. Ett par artiklar kan ge lite mer kunskap. Utförlig information finns på gnunet.org* .

Ladda ner GNUnet

Du hittar GNUnet på ftp://ftp.gnu.org/gnu/gnunet/* , GNU:s speglingar och på gnunet.org* .

Krav
  • GNU/Linux, FreeBSD, OpenBSD, NetBSD, OS X, Windows XP or Solaris.
  • GNU MP Bignum Library (>= 4.0.0)
  • libgcrypt (>= 1.2.0)
  • gtk+ (>= 2.4)
  • libextractor (>= 0.4.2)
Kom ihåg att detta är en betaversion. Det kan fortfarande förekomma problem med stabilitet, skalbarhet och prestanda.
Nätverket är ännu mycket litet. Väldigt lite innehåll är ännu tillgängligt. Det kan vara en god idé att prenumerera på GNUnet:s epostlistor.
