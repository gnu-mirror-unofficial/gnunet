Introduzione

GNUnet è un framework per comunicazioni peer-to-peer sicure. The primary application implemented within the GNUnet framework is anonymous censorship-resistant file-sharing.*

The goal of the GNUnet project is to provide an infrastructure for secure peer-to-peer networking.* All communication in GNUnet is authenticated and link-to-link encrypted.* The economic model makes attacks on the network harder since the economics can be used to control resource usage.* Gli utenti di GNUnet scambiano messaggi grazie ad un modulo intercambiabile astraente il servizio di trasporto. Attualmente sono disponibili servizi basati su UDP, TCP, HTTP ed SMTP. The GNUnet core provides mechanisms to perform resource allocations for CPU, bandwidth and storage space.* The core enforces resource limitations set by the user.* GNUnet non dipende da alcun servizio centralizzato. Dalla infrastruttura di base di GNUnet possono essere sviluppati nuovi protocolli paritetici. Tra i protocolli attualmente esistenti vi sono la condivisione anonima di file, un semplice protocollo di discussione, la tachimetria della trasmissione di messaggi, e la visualizzazione della topologia della rete.
Anche se i nostri obiettivi sono simili a progetti come Freenet, Gnutella, Mnet e altri, speriamo di mettere a disposizione dei nostri utenti una combinazione superiore di caratteristiche per utenti che privilegiano la sicurezza sopra alla efficienza.

È disponibile un elenco di domande frequenti . Una coppia di articoli per approfondire. Documentazione ulteriore è disponibile su gnunet.org.

Scaricare GNUnet

Puoi trovare GNUnet su ftp://ftp.gnu.org/gnu/gnunet/, il mirror GNU o su gnunet.org.

Requisiti
  • GNU/Linux, FreeBSD, OpenBSD, NetBSD, OS X, Windows XP or Solaris.
  • GNU MP Bignum Library (>= 4.0.0)
  • libgcrypt (>= 1.2.0)
  • gtk+ (>= 2.6)
  • libextractor (>= 0.5.4)
Ricorda che questa è una versione Beta. Potrebbero ancora essere presenti problemi di stabilità, scalabilità, o prestazioni.
