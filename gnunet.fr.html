Introduction

GNUnet est un framework pour reseautage pair a pair securitaire. La premiere application implementee dans GNUnet est un systeme de partage de fichier anonyme et resisant a la censure

Le but du projet GNUnet est de fournir une infrastructure pour un réseau pair à pair sécurisé. Toutes les communications dans GNUnet sont authentifiées et cryptées de lien en lien. The economic model makes attacks on the network harder since the economics can be used to control resource usage.* Les pairs sur GNUnet echangent des messages grace a un module remplacable abstrayant le service de transport. Actuellement, des services de transport basés sur UDP, TCP, HTTP et SMTP sont disponibles. Le noyau GNUnet fournit des mécanismes pour réaliser les allocations de ressources en CPU, bande passante et espace de stockage. Le noyau garantit le respect des limitations de ressources fixées par l’utilisateur. GNUnet ne repose sur aucun service centralisé. New Peer-to-Peer protocols can be easily implemented on top of the basic GNUnet infrastructure.* Les protocoles actuels sont le partage de fichiers anonyme, un protocole de discussion rudimentaire, des tests de rapidité du débit des messages et la visualisation de la topologie du réseau.
While our goals are similar to projects like Freenet, Gnutella, Mnet, and others, we hope to provide a superior combination of features for users that value security more than efficiency.*

Une liste de questions fréquemment posées est disponible. Quelques articles peuvent apporter un aperçu supplémentaire. La documentation exhaustive peut être trouvée sur gnunet.org.

Téléchargement de GNUnet

GNUnet peut être trouvé sur ftp://ftp.gnu.org/gnu/gnunet/, sur les miroirs GNU ou sur gnunet.org.

Outils nécessaires
  • GNU/Linux, FreeBSD, OpenBSD, NetBSD, OS X, Windows XP or Solaris.
  • GNU MP Bignum Library (>= 4.0.0)
  • libgcrypt (>= 1.2.0)
  • gtk+ (>= 2.4)
  • libextractor (>= 0.4.2)
Please keep in mind that this is a beta version.* There may still be some stability, scalability and performance issues.*
So far the network is very small.* Très peu de contenu est disponible pour le moment. Une inscription aux listes de diffusion de GNUnet peut être une bonne idée.
